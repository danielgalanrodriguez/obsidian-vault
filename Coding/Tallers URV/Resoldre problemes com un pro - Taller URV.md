# Exercici
Objectiu:
Acabar el projecte SIE

Preguntes:
1. Per a que el vols aconseguir el teu objectiu?
	- Per a millorar el sistema (el codi). Per apendre. Actualitzarlo y evitar pagar per la versio antiga PHP
2. per millorar el codi, per apendre i per evitar pagar el PHP
3. per a millar el codi, esta fet una me. Apendre a l'hora, aixi faix les coses millor. Per actialitzar el PHP. No pagar tant.
4. Per a millorar la qualitat del codi, esta fet una merda. Per a apendre durant el cami. Has apres coses noves y saps com fer millor les coses. Actualitzar PHP
5. Per a millorar la qualitat del codi. Tambe la usabilitat, estan fetes una me. PEr actualitzar el PHP
6. Per a actualitzar el PHP i no pagar, per a eliminar el fallo de seguretat.

Conclusió: el problema real no es el que hi ha a la superficie. Esta mes al fons. 

> No confonguem el simptoma amb l'esencia del problema.


# Diagrama ishikawa

6 aspectes.

1. Persones: analitza les persones que resoldran el problema
2. Material: Mira si el material que utilitzes 
3. Recursos Tecnics: Analitza les eines que utilitzaras i mira si realment et serveixen
	- Tin accés als recursos que necesito?
4. Métode: tenim algun proces? Quin metode fas servir per solucionar el problema
	 - Estas fent les coses des de 0 tot el rato i podries estandaritzar
5. Entorn: el problema esta situat en un entorn fisic. Circumstancies externes. El lloc on ets. Com afecta?
6. Indicadors: metricas. com medeixes l'evolucio del tema

## Diagrama aplicat al objectiu

1. Soc jo, per tant es posible aconseguir l'objectiu. Pero tin temps i energia limitad, ningu m'ajuda. I porto tant de temps en aixo que he perdut la motivacio. Hi ha risk de fallida.
2. El material es correcte, les llibreries que utilitzo estan actualizades i ben mantingudes.
3. Tinc accès a tots el recursos que necessito.
4. El proces falla. Suposo que haure de assignar unes hores o un horari per posrhi temps i acabarlo. Pero clar depeng molt de terceres persones i aixo fa dific posar un horari concret.
5. A casa hi han distractions, les terceres persones. He de mirar de sortir per
6. Anar acabant l'aplicacio.
