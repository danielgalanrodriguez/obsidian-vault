src: [https://brew.sh/](https://brew.sh/)

# Installation
Installed by just running one command in the terminal.
Visit the website to get the latest command instructions: https://brew.sh/

Command at DEC-2025:
``` zsh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```


Installation path: `/opt/homebrew`
Binaries path: `/opt/homebrew/bin/`
