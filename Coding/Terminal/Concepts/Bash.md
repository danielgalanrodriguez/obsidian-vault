# Bash - AKA (Bourne Again Shell)

# What it is:
A specific **implementation** of a Shell.
   
# Function: 
It's a widely used, standardised command interpreter known for its **scripting portability** across many systems. It was the default shell on macOS until 2019.
   
# Analogy:
 A **brand/model of the operating system** (e.g., "Android" or "Linux OS").

# Interesting fact
Bash is an enhanced, open-source successor to the original **Bourne Shell (`sh`)**. It introduced features like command history, job control, and better scripting capabilities.
