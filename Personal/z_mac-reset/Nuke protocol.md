# Mac OS settings
- Tweak finder settings:
	- Show path in finder: finder -> view -> show path bar
- Enable "tap to click" : 
	- System Settings -> Trackpad -> Point & Click tab
	- toggle on Tap to click
- Arrange touch bar buttons:
	- System Settings -> Keyboard -> Touch bar settings -> Customise Control Strip
- Add shortcut to arrange windows
	- System Settings -> Keyboard -> Keyboard Shortcuts -> App shorcuts
		- Window->Move & Resize->Left
		- Window->Move & Resize->Right
		- src: https://www.reddit.com/r/MacOS/comments/1fl7ffu/how_to_change_sequoia_window_tiling_shortcut/
		- ![[mac_tiling_shortcut.png]]

- Set battery settings: System settings-> Battery
	- Set low power mode
	- Set -> Options...
- Set menu bar settings: System settings -> Menu Bar
	- Battery Options -> Show percentage
	- Clock Options:
	- ![[Screenshot 2025-12-31 at 18.28.54.png]]

- Copy Music folder inside the documents folder:
	- Import library (inside '\_meta\_' folder) [instructions](https://discussions.apple.com/thread/253777058?sortBy=rank)
	- Update media folder Music->Settings->files
	- Uncheck, copy files to media folders
- Setup mail accounts in mail app
	- gmail
	- hotmail
